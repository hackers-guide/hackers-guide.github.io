Hosting Resources

Free (or mostly free) ways to host your hack

Here's a list of free resources to host your hack site

Heroku

Heroku is simple, easy, and super popular. You'll likely be able to get help from the hackers around you pretty easily.

Highlights

  • Free
  • No server knowledge necessary
  • No low level configuration required for most apps
  • Easy integrations with external services
  • Command line deployment

Platforms

  • Ruby
  • PHP
  • Node.js
  • Pythjon
  • Java
  • Clojure
  • Scala

Get started here.

Amazon Web Services (AWS)

AWS supplies some pretty amazing point and click options for free, or nearly free, hosting of anything.

Highlights

  • One year of compute hours on the free platform
  • No special prep
  • Point and click or roll your own
  • Web based management
  • Command line management
  • Visual Studio Integration

Platforms

  • Point and Click with Elastic Beanstalk.
    • Ruby
    • PHP
    • Node.js
    • Pythjon
    • Java
  • Visual Studio Integration for .NET with Elastic Beanstalk
  • Roll your own for any Linux or Windows hosted application on Elastic Compute Cloud (EC2).

Get started here for elastic beanstalk or here for Elastic Compute Cloud (EC2).

Azure Web Services

Azure Web Services is easy enough but not terribly popular outside of .NET.

Highlights

  • $200 free
  • SourceControl integration for continuous deployment
  • No special prep
  • Point and click
  • Web based management
  • Command line management
  • Visual Studio Integration

Platforms

  • .NET
  • Ruby
  • PHP
  • Node.js
  • Pythjon
  • Java

Get started here.

Compose

No-SQL hosted databases for free

Highlights

  • Free for sandbox environment
  • Super easy to set up and use
  • No special prep
  • Point and click
  • Web based management

Platforms

  • MongoDB
  • Elasticsearch
  • RethinkDB
  • Redis

Get started here.
